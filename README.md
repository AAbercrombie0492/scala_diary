# scala_diary
Archiving my ascent up the stairway to Scala heaven.  

## Setup
1. Setup Scala
  A) Download
  ```bash
  wget https://downloads.lightbend.com/scala/2.12.2/scala-2.12.2.tgz
  ```
  B) Extract Scala
  ```bash
  tar -xvzf scala-2.12.2.tgz
  ```
  C) Export PATH
  ```bash
  export PATH="~/projects/scala_diary/scala-2.12.2/bin:$PATH"
  ```
